# <span style="color: green;">1. Product Overview</span>

InHand Connect Service (InConnect) is a one-stop connectivity service for enterprise customers.  
With a user-friendly interface and simplified workflows, together with InHand networking devices, it helps enterprises quickly connect global business sites, enable interconnection between devices and information, and support IT modernization and digital transformation.

InConnect allows IT engineers to remotely access terminals connected downstream of routers/gateways for remote maintenance, program deployment, and building data channels for continuous terminal data reporting to business centers. Supported terminals include industrial PCs, servers, cameras, PLCs, HMIs, RTUs, controllers, and more.

# <span style="color: green;">3. Key Advantages</span>

- Fast and cost-effective remote network deployment: No public IP lines or dedicated hardware VPN firewalls are required. InHand devices can connect through 4G LTE or wired networks.
- Multi-end interconnectivity and access policy control: Supports remote access and control of on-site devices, while secure gateway mechanisms can restrict inter-device communication as needed.
- Automated network deployment: After routers and downstream devices are added, secure connections are established automatically, supporting smart IP mapping and real-IP communication.
- Data encryption and transmission security: Cloud servers combined with advanced access control and multi-layer encryption logic ensure network security.
- Multi-terminal access support: Supports Windows, Linux, Android, and iPhone clients, as well as terminal access for PLCs, HMIs, cameras, and more.
- One-stop service: Integrated networking devices and platform with cloud deployment, no local O&M required, and elastic scalability.

# <span style="color: green;">4. Product Features</span>

## <span style="color: green;">Site Management</span>

| Feature Item | Description |
|---|---|
| Router/Gateway | Asset management for serial number, ICCID, IMSI, etc.; operational monitoring such as network status, traffic consumption, and signal history |
| Terminal Devices | Centralized management and maintenance for PLC/HMI and other terminal devices, with remote access support |
| Configuration Update | Batch remote configuration and task delivery management |
| Firmware Upgrade | Batch firmware upgrade and upgrade task management |
| Remote WEB | Remote access to routers for configuration changes |
| Operations Analysis | Carrier traffic statistics, connection statistics, and GIS device location tracking |
| Task Tracking | Management of all device system tasks, with online tracking of status and progress |

## <span style="color: green;">Network Management</span>

| Feature Item | Description |
|---|---|
| Client Access | Supports multiple clients, user login, and access duration control; restricts accessible MAC addresses |
| Peer-to-Peer Network | Designed for remote device maintenance, supports maintenance for PLC/HMI and other devices, with virtual IP and real IP access |
| Star Network | Designed for remote office scenarios, supports access to internal enterprise servers and file resources from home or while traveling |
