# <span style="color: green;">1. Product Overview</span>

Device Manager is a next-generation IoT device management cloud platform independently developed and operated by InHand Networks. Through a visual user interface and simple workflows, the Device Manager platform enables users to conveniently manage and monitor InHand hardware devices such as routers and gateways, achieve rapid device integration, and perform one-click management of massive device fleets. Cloud deployment provides an easy "use-and-go" experience, allowing users to focus on core business and empowering enterprise growth.

## <span style="color: green;">Core Value</span>

- **Rich Device Management**: Device Manager provides rich capabilities suitable for a wide range of industry scenarios.
- **Worldwide Remote Management**: Manage InHand hardware devices distributed globally without leaving your office, saving time and effort.
- **Multi-Role Permission Control**: Allows IT and OT teams to collaborate efficiently from anywhere.
- **MQTT Protocol Access**: Technical architecture supports MQTT for broad IoT applications.
- **Low-Bandwidth & Massive Scale**: Suitable for field environments with low bandwidth and high latency; supports massive device access and cloud-based management.
- **RESTful API Integration**: Provides RESTful API interfaces for secondary development and rapid integration with other service systems.
- **End-to-End Security**: CA certification, data encryption, user authentication, and access control, together with multiple integrated security mechanisms on both platform and devices, ensure network and data transmission security.
