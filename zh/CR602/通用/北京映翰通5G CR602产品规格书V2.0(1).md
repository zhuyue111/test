## <span style="color: green;">1. 产品概述</span>

CR602是一款创新的5G紧凑型蜂窝路由器，专为商铺门店、小企业办公、临时工作、户外出行、家庭娱乐场景设计的高性能移动网络接入设备。产品将先进的5G蜂窝连接技术、Wi-Fi 7无线技术、大容量电池等丰富功能集成于紧凑的外壳内，为用户提供随时随地的高速网络连接解决方案。

CR602采用一体化紧凑设计，支持三SIM（双 Nano + 预置 eSIM）智能切换，配备高性能5G模组和Wi-Fi 7无线技术，能够为各种室内外场景提供高速稳定的网络连接。设备支持InCloud云管理平台，实现远程设备管理和运维，大幅降低运维成本。

### <span style="color: green;">5.2 SIM 卡管理</span>

#### <span style="color: green;">5.2.1 三 SIM 槽硬件配置</span>

- **物理形态**：2× Nano-SIM + 1× 内置 eSIM（GSMA Consumer 规格，出厂预置固定运营商 Profile）
- **工作模式**：单卡在线，其余作为热备，秒级故障切换
- **eSIM 优势**（基于预置运营商 Profile）：
  - **免插拔、免等待**：出厂预激活，开机即联网，无需等待实体 SIM 卡邮寄与开卡
  - **高可靠性**：无机械触点与卡槽，抗振动、抗腐蚀，适配移动与户外场景
  - **高集成度**：节省卡槽空间，腾出空间用于电池容量、天线布局优化
  - **提升防护**：减少卡槽开孔，整机防尘、防潮表现更佳
  - **简化产线**：免去 SIM 卡组装与上电测试工序,提升出厂一致性
  - **参数远程运维**：虽不支持动态切换 Profile，但 APN / 网络优先级 / 漫游策略等参数可通过 InCloud 云平台远程下发更新

#### <span style="color: green;">5.2.2 切卡优先级灵活配置</span>

支持对三张 SIM 自定义角色与优先级，三个槽位可**任意指派**为主卡 / 备卡 1 / 备卡 2：

| 角色          | 说明                                            |
| ------------- | ----------------------------------------------- |
| **主卡（Primary）**   | 首选拨号 SIM，开机 / 重启后默认使用             |
| **备卡 1（Secondary）** | 主卡异常时的第一切换目标                       |
| **备卡 2（Tertiary）**  | 备卡 1 仍不可用时的最后兜底                    |

- **三种配置入口**：Web 本地配置、移动 APP、InCloud 云端批量下发
- **优先级动态切换**（例如：主卡 A流量预警之后切运营商 B）

#### <span style="color: green;">5.2.3 切换触发条件</span>

任一条件触发即按优先级切换至下一张可用 SIM：

| 触发类别     | 触发条件                              | 可配参数                          |
| ------------ | ------------------------------------- | --------------------------------- |
| **拨号失败** | SIM卡异常，PDP 激活失败等                  |                       |
| **链路探测** | ICMP 探测连续超时3次         | 目标地址、超时阈值      |
| **流量阈值** | 当月已用流量达到设定上限              | 流量上限、计量周期、告警提前量    |

#### <span style="color: green;">5.2.4 每卡独立配置</span>

- 独立 APN / 用户名 / 密码 / 认证类型（Auto / PAP / CHAP）
- 独立网络协议栈 (IPv4 / IPv6 / IPv4&IPv6)
- 独立网络制式偏好（5G SA / 4G&5G / 4G / Auto）
- 独立流量用量统计与上限告警

### <span style="color: green;">5.3 有线网络</span>

- **网口接口**：2× 2.5G Gigabit RJ45，支持WAN/LAN自适应
- **USB接口**：1× USB-C 2.0，支持充电和固件升级
- **IPv6支持**：支持IPv6协议
- **数据吞吐量**：最大2Gbps

### <span style="color: green;">5.4 Wi-Fi 7无线接入</span>

- **标准协议**：IEEE 802.11be（Wi-Fi 7）
- **工作频段**：2.4GHz + 5GHz 双频并发
- **无线规格**：BE3000
- **最大无线并发**：32 台
- **覆盖距离**：Wi-Fi 有效覆盖距离 50 米（无遮挡参考值）
- **MIMO技术**：3×3 MIMO
- **信道宽度**：5GHz 支持 20/40/80/160MHz；2.4GHz 支持 20/40MHz
- **调制技术**：支持4K QAM
- **安全加密**：WPA3-PSK/WPA2-PSK

### <span style="color: green;">5.5 接入容量</span>

| 接入类型 | 最大并发数 | 说明 |
| -------- | ---------- | ---- |
| **无线接入（Wi-Fi）** | ≤ 32 台 | 2.4 GHz + 5 GHz 双频并发，受 Wi-Fi 7 BE3000 规格上限约束 |
| **有线接入** | ≤ 96 台 | 通过 2× 2.5G 网口下接交换机扩展 |
| **整机合计** | **≤ 128 台** | 受 DHCP 地址池与 NAT 会话上限约束 |

> 实际并发能力受单台终端流量特征、应用类型与无线信道质量影响，建议根据业务模型预留余量。

## <span style="color: green;">8. 安全特性</span>

### <span style="color: green;">8.1 防火墙</span>

- **状态检测**：基于连接状态的数据包检测
- **入站规则**：精细化入站访问控制
- **出站规则**：出站流量策略管理
- **端口转发**：支持端口映射与内网穿透
- **NAT功能**：SNAT/DNAT，支持NAPT
- **MAC过滤**：基于MAC地址的访问控制
- **域名过滤**：基于域名的内容过滤
- **局域网防火墙**：东西向流量安全隔离

### <span style="color: green;">8.2 策略路由</span>

- **智能分流**：基于源/目的地址、端口、协议的策略路由

### <span style="color: green;">8.3 流量整形</span>

- **带宽限制**：接口级带宽控制
