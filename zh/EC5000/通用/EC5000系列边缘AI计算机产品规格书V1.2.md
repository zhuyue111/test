# <span style="color: green;">1. 产品概述</span>

**EC5000 是基于 NVIDIA Jetson Orin 的工业级 AI 边缘计算机，面向高性能视觉与推理场景，支持本地智能与云端协同。**


**产品特点：**
- **高性能AI:** 预装 Orin Nano/NX，提供 40TOPS/100TOPS 可选算力
- **接口丰富:** 双网口、双串口、DI/DO、CAN FD、USB3.2、HDMI、音频与GMSL可选
- **扩展灵活:** 支持 M.2 B-Key（蜂窝）、E-Key（Wi-Fi/BT）、M-Key（NVMe）
- **可靠稳定:** 双SIM、链路检测、看门狗、无风扇工业设计
- **云端运维:** 支持 DeviceLive 远程管理与 DSA 数据采集上云

## <span style="color: green;">核心技术指标</span>

| 技术指标 | 规格 |
|---|---|
| AI平台 | NVIDIA Jetson Orin Nano / Orin NX |
| AI算力 | 40TOPS / 100TOPS（按型号） |
| 云端运维 | DeviceLive 远程管理 |
| 数据采集 | DSA 数据采集上云 |
| 蜂窝网络 | 5G / 4G CAT6 / 无蜂窝（按型号） |
| 双SIM与链路可靠性 | 双SIM、链路检测、看门狗 |
| 处理器 | Cortex-A78AE（6核/8核，按型号） |
| 内存 | 8GB（EC5350）/16GB（EC5550） |

| 扩展能力 | M.2 B-Key（蜂窝）、E-Key（Wi-Fi/BT）、M-Key（NVMe） |
| 工作温度 | -20~60℃（0.7m/s风速） |
| 防护等级 | IP30 |
