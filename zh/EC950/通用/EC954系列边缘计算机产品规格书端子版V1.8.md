# <span style="color: green;">1. 产品概述</span>

**EC954 是高性能多接口 AI 边缘计算机，支持边缘智能推理、协议采集与云端运维，适用于工业复杂场景。**


**产品特点：**
- **AI可扩展:** 内置NPU并支持 Hailo-8 扩展，算力最高可达 26 TOPS
- **接口密集:** 4×GE、12路串口、CAN、I/O、mSATA、USB、HDMI
- **通信可靠:** 有线/蜂窝/Wi-Fi 互备，双SIM切换与链路自愈
- **安全加固:** Secure Boot、TPM2.0、TrustZone、防火墙与VPN
- **开放运维:** Debian + Docker + DSA + DeviceLive 云管平台
