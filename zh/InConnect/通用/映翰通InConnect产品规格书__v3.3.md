# <span style="color: green;">1. 产品概述</span>

InHand Connect Service（简称 InConnect）是面向企业客户的一站式联网服务。  
通过人性化界面与简化流程，结合映翰通联网设备，帮助企业快速连接全球业务网点，实现设备与信息互联互通，支持企业信息化建设与数字化转型。

InConnect 支持 IT 工程师远程访问路由器/网关下连终端，进行远程维护、程序上下载，以及建立终端数据持续上报到业务中心的数据通道。可访问终端包括工控机、服务器、摄像机、PLC、HMI、RTU、控制器等。

# <span style="color: green;">3. 特性优势</span>

- 快速低成本构建远程网络：无需公网 IP 线路和硬件 VPN 防火墙，使用映翰通设备接入 4G LTE 或有线网络即可。
- 多端访问互通与访问策略控制：支持远程访问与控制现场设备，并可通过网关安全机制限制设备间互通。
- 自动化网络部署：添加路由器与下端设备后可自动建立安全连接，支持智能 IP 映射与实 IP 通信。
- 数据加密与传输安全：云端服务器结合高级访问控制与多层加密逻辑，保障网络安全。
- 多类型终端接入：支持 Windows、Linux、Android、iPhone 客户端，支持 PLC、HMI、摄像头等终端接入。
- 一站式服务：联网设备与平台集成，云端部署无需本地运维，支持弹性扩展。

# <span style="color: green;">4. 产品功能</span>

## <span style="color: green;">站点管理</span>

| 功能项 | 说明 |
|---|---|
| 路由器/网关 | 序列号、ICCID、IMSI 等资产管理；网络状态、流量消耗、信号历史等运行监控 |
| 终端设备 | 集中管理维护 PLC / HMI 等终端设备，支持远程访问 |
| 配置更新 | 批量远程配置、管理下发任务 |
| 固件升级 | 批量固件升级、管理升级任务 |
| 远程 WEB | 远程访问路由器并修改配置 |
| 运行分析 | 运营商流量统计、连接统计、GIS 设备位置跟踪 |
| 任务跟踪 | 管理所有设备系统任务，在线跟踪状态与进度 |

## <span style="color: green;">网络管理</span>

| 功能项 | 说明 |
|---|---|
| 客户端接入 | 支持多客户端连接、用户登录及访问时长控制；限制访问 MAC 地址 |
| 对等网络 | 面向远程设备维护场景，支持维护 PLC/HMI 等设备，支持虚拟 IP 与实 IP 访问 |
| 星型网络 | 面向远程办公场景，支持在家或外地访问公司内网服务器和文件资源 |
| 网络权限 | 划分网络组，控制用户对站点的访问权限及账号数据管理权限 |
| 流量统计 | 统计远程连接流量消耗 |

## <span style="color: green;">网络接入能力</span>

| 项目 | 规格 |
|---|---|
| 路由器接入上限 | 单个注册账号最大接入点 4000 个 |
| 终端设备接入上限 | 单个现场路由器节点最多支持 254 个终端设备接入 |
