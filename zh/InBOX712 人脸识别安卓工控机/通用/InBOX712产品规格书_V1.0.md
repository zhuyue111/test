# <span style="color: green;">1. 产品概述</span>

**InBOX712 系列面向智慧商业领域，搭载瑞芯微6核处理器，支持双屏异显与4G智联。**

**产品特点：**

- **强劲性能:** 瑞芯微6核1.8GHz，2GB/16GB存储，双HDMI双屏异显
- **4G智联:** 支持4G全网通，随处可得不间断的互联网接入
- **丰富接口:** 3路RS232、5路USB、千兆网口、TypeC接口
- **工业品质:** IP40防护、Level 3 EMC、-20°C~70°C宽温工作
- **安卓生态:** Android 7.1系统，兼容丰富应用软件与远程升级
